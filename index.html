YouLearn App

this is a mobile application that allows viewers to play Youtube videos instead of watching them,This application has 2 main purposes:

It extracts the transcript of a Youtube video and summarizes it.

It generates a quiz from the processed text in order to give users the possibility to learn faster.

Start Screen

Home Page

by clicking on "summarize" of the video the django backend will:

1. Extract the transcript ( Youtube API )

2. Grammatical and spelling correction

3. Divide the text according to the time period of each chapter, The start and end time of each chapter is extracted from the video description (there is no Youtube API for this).

4. Summarize the text for each chapter using the Gensim library. The summary is based on the ranking of the sentences in the text using a variation of the TextRank algorithm

Summarize 2

This application generates only objective questions (to choose the right answer among several possibilities).

1. It selects a sentence from a specific chapter.

2. Convert the text into a list of sentences, then the sentences are converted into a list of words.

3. Separate the names of the created word list and choose an appropriate name to hide

4. Generate similar names for the hidden name using the "wordnet" package of nltk.corpus
